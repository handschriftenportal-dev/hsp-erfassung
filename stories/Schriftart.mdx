export const ExampleText = "Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua."
export const ExampleSpecialCharacter = Object.fromEntries(
  SonderzeichenAPI.gruppen().map((gruppe) =>
    [
      gruppe,
      SonderzeichenAPI.getList({ gruppe })
        .map(key => SonderzeichenAPI.getSonderzeichen(key).sign)
        .join(' ')
    ]
))

export const unicodeRange = (from, to) => {
  let result = []
  for (let i = from; i <= to; i++) {
    result.push(String.fromCodePoint(i))
  }
  return result.join(' ')
}

## Schriftart für das Handschriftenportal

Da Beschreibungen von Handschriften alte Texte mit speziellen Sonderzeichen
benötigen, sind die Anforderungen an eine einheitliche und gut aufeinander
abgestimmte Schriftart hoch. Es können Texte in unterschiedlichen Sprachen
aus unterschiedlichen Zeitaltern auftreten.

Um alle nötigen Schriftzeichen zu untersuchen nutzen wir Adobe NotDef, um
fehlende Schriftzeichen schneller zu erkennen.

## Außerhalb des Editors

Für Bedienelemente, Menüs, Hoveransichten, Buttons, Hilfetexte, inklusive
Sidebar

- **Font:** Roboto
- **Weights:** 400(regular), 700(bold)

## Innerhalb des Editors

Für Texte innerhalb einer Beschreibung, inklusive Vorschautexten in der Sidebar

- **Font:** Junicode Beta 2
- **Fallback:** Adobe Not-Def
- **Weights:** 400(regular), 700(bold)

### Geomantische Figuren

Die geomantischen Figuren sind vorläufig in Unicode als _proposal_ aufgenommen.
Gegebenenfalls sind sie 2025 in Unicode Version 17.0.0 enthalten (siehe
<a href="https://www.unicode.org/L2/L2023/23231.htm#177-C34"
   target="_blank">L2/L2023/23231.htm#177-C34</a>).
Eine Schriftart, welche die geomantischen Figuren unter diesen Codepoints
speichert, muss noch gefunden oder erstellt werden.
