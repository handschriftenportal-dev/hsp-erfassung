## Einführung Storybook

### Was ist Storybook?

Storybook ist ein Tool zur Dokumentation von UI-Komponenten innerhalb einer
Webanwendung. Es kann dabei auf den Quellcode zugreifen, mit JavaScript einfach
erweitert werden, unterstützt selbstgeschriebene und automatische Docs mit
typischen Nutzungsszenarien.

Weiterhin bietet es Möglichkeit zur Testintegration von E2E-Tests.

### Struktur

In dem Menü finden sich:
- die Definition von Farben
- die im System genutzten Icons
- eine Übersicht der Schriftart und der notwendigen Sonderzeichen.

Darunter findet sich das Design System, in dem die vom Editor genutzten
Komponenten aufgelistet und ergänzt werden können.

In der oberen Leiste gibt es die Möglichkeit in den Stories:
- die Hintergrundfarbe anzupassen
- Ein Grid ein- und auszublenden
- Die Sprache der i18n-Datei der Komponenten zu verändern
- Sich Rahmen innerhalb der Komponente anzeigen zu lassen
